Telegram Bot
============

The Telegram Bot acts as a link between the user and the rest of the system. It is responsible for serving the interface through the Telegram application and collecting the text and images that the user enters to analyze this entry in Dialogflow and to identify the user's intention along with the keywords of each of the phrases, in addition to showing the responses generated by the system through the interface.
